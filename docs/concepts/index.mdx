# Concepts

Polkadot Live is striving to become an essential productivity and management tool for users of Polkadot and Substrate based networks. The core of Polkadot Live involves enabling subscriptions and receiving notifications when certain events occur on Substrate based blockchain networks. Notifications can be delivered as both native OS notifications, as well as event items rendered in the main window.
